# WebCrawler

<b>Flask</b>, <b>RoboBrowser</b> and <b>BeautifulSoup</b> are required to be able to run the program. <br>
It's take around <b>5 seconds</b> to generate the result for a single ad page, every page has around 45 ads. <br>
